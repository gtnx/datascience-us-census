# Analyse du dataset Us Census

## Installation

    git clone git@github.com:gtnx/datascience-us-census.git ${repo}
    cd {repo}/data
    wget http://thomasdata.s3.amazonaws.com/ds/us_census_full.zip
    unzip us_census_full.zip
